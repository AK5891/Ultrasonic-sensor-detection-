# Ultrasonic-sensor-detection

This is a basic arduino set up that uses an ultrasonic sensor in order to detect objects within a certain range. Once an object is detected, an alarm is set off. This was coded in C++ on Arduino IDE. 

# Requirements

1. Make sure to have Arduino IDE installed

```
2. Install the HC-SR04 library for Arduino Uno
```
# Parts Needed
x1 Arduino Uno

x1 HC-SRO4 ultrasonic sensor

x6 male to male jumper wires

x1 passive buzzer

x1 breadboard
